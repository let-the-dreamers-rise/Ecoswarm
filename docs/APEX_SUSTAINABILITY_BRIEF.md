# Apex Sustainability Brief

## 100-Word Description

EcoSwarm Regen is a Hedera-native milestone payout treasury for community climate projects. Local operators submit proof-backed projects, the platform converts them into sponsor-ready release cases, and a semi-autonomous agent network triages verification, authorization, settlement, and reputation updates. AI ranks funding opportunities, HTS-linked impact receipts track outcomes, and HCS-backed checkpoints keep the audit trail visible. The result is a deployable coordination layer for restoration and resilience funding: faster trust, clearer proof, repeatable releases, and a real operating system for sponsors, verifiers, and local operators instead of a static sustainability dashboard.

## Why It Fits Track 3

- It is a financial system for ecological and social impact.
- It includes an on-chain verification pattern through proof hashes, receipts, and Hedera event records.
- It is community-driven because projects originate from local operators and cooperatives, not top-down institutions.
- It behaves like a marketplace by ranking payout cases and matching treasury capital to the strongest opportunities.

## 60-Second Demo Script

1. Open `Overview` and position EcoSwarm Regen as a milestone payout treasury for community climate projects.
2. Start simulation and focus on one flagship case entering the queue with proof confidence, urgency, and named counterparties.
3. Open `Case Room` and show the shared sponsor, verifier, and operator workflow around one payout case.
4. Open `Agent Network` and dispatch the recommended agent step for the same case.
5. Open `Blueprint` and show the wedge, pilot contract value, projected Hedera usage, and rollout plan.
6. Open `Client Portal` and show synchronized sponsor alerts for the same case.
7. End on `Audit Trail` and the Hedera event stream to prove the flow is auditable end to end.
